# Android_Weather
Personal Android App Project, offered by Udacity [Developing Android Apps: Android Fundamentals](https://www.udacity.com/course/ud853).

Features added to far:
* Fetching realtime weather data from http://openweathermap.org
* Preference (location, units)
* Having multiple screen (activity)
* Share
* Database testing using AndroidTestCase
* Content Provider
* Cursor Loader
* Updated UI
* Multi-pane support on bigger screen
* Notification support + on/off preference
